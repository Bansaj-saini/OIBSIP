### Retail Sales Data Analysis (EDA)

# Project Overview
This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to uncover patterns, trends, and business insights.

# Objective
- Analyze sales data
- Understand customer behavior
- Generate actionable business recommendations

# Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

### Key Steps Performed

# 1. Data Loading & Cleaning
- Loaded dataset using pandas
- Checked for missing values
- Verified data types

# 2. Descriptive Analysis
- Calculated mean, median, and standard deviation
- Analyzed numerical features

# 3. Time Series Analysis
- Plotted monthly sales trends
- Identified peak sales months

# 4. Customer Analysis
- Gender distribution
- Age group analysis

# 5. Product Analysis
- Category-wise revenue
- Top-performing products

# 6. Correlation Analysis
- Heatmap to find relationships between variables

# 7. Additional Insights
- Scatter plots to identify hidden patterns

# Key Insights
- Sales show seasonal trends
- Middle-aged customers dominate purchases
- Small transactions are more frequent

# Business Recommendations
1. Increase stock during peak months
2. Target middle-aged customers
3. Offer bundle discounts to increase sales

# Dataset
- Retail Sales Dataset (Kaggle)

# Conclusion
EDA helped in identifying important trends and patterns which can improve business decision-making.
